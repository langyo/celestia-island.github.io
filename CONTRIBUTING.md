# Contributing to Celestia Island Homepage

## Commit Convention

All commits MUST follow this format:

```
<gitmoji> <imperative sentence in English, lowercase, no period>
```

### Rules

- Start with a gitmoji from [gitmoji.dev](https://gitmoji.dev)
- One short imperative sentence in English
- No prefixes (no `feat:`, `fix:`, etc.)
- No version numbers
- No trailing period
- No additional blank lines or body text unless truly necessary

### Examples

- `🚀 initial landing page with Vue3 and Three.js`
- `✨ add French and Russian translations`
- `🐛 fix snap scroll on Safari`
- `💄 improve project card hover animation`
- `♻️ refactor i18n to JSON format`
- `📝 add contributing guidelines`
- `🔧 update Vite to v7`

### Reference

| Gitmoji | Use when |
|---------|----------|
| 🎨 | Improve structure / format |
| ⚡️ | Improve performance |
| 🔥 | Remove code / files |
| 🐛 | Fix a bug |
| 🚑 | Critical hotfix |
| ✨ | Introduce new features |
| 📝 | Add or update documentation |
| 🚀 | Deploy / init stuff |
| 💄 | Add or update UI / style |
| ♻️ | Refactor code |
| 🔧 | Add or update configuration |
| ⏪️ | Revert changes |
| 💡 | Add or update comments |
| 🌐 | Internationalization |
