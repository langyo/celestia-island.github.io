# Entelecheia

**ハーネスエンジニアリング**の原則に基づいて構築された、安全クリティカルな産業制御向けのマルチエージェントプラットフォーム。

AIモデルに無制限のツールアクセスを与えるのではなく、Entelecheiaは各エージェントをその専門領域に制約します——何ができて何ができないかを正確に定義します。LLMが見るのは少数のプリミティブのみ。実際のあらゆる動作はIEPL TypeScriptパイプライン内で実行され、多数のエージェントにまたがる広大なMCPツール群へディスパッチされます。

際立つのはその産業の視点です。クロスベンダーのプロトコル対応（Modbus、S7comm、OPC UA）、命令レビューから人の承認までの多層安全ゲート、そしてコンテナ隔離実行。制御室から工場現場まで、Entelecheiaが調整を担い、エージェントは常に境界の内側で行動し——決して越えません。
