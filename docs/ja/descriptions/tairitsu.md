# Tairitsu

**WASIネイティブ**のフルスタックWebフレームワーク。アプリケーション全体——フロントエンドUIとバックエンドロジック——をRustで記述し、WASIコンポーネントにコンパイルして、ブラウザ、サーバー、エッジにデプロイします。

JavaScriptのグルーコードや`wasm-bindgen`に依存する従来のWebAssemblyアプローチとは異なり、TairitsuはWebAssembly Component Modelを通じてWASIを直接ターゲットとします。フロントエンドとバックエンドは、型付きインターフェースを持つ単一のポータブルランタイムを共有します。一つのランタイム、一つの言語、一つのコードベース。
