# Noa

AIネイティブな分散バージョン管理システム — ソースコードは git、AIエージェントのイテレーションデータは noa。

同一ワーキングツリー内で `.git` と共存。エージェントごとの追記専用 JSONL ログ（ゼロロック）により、数百のエージェントが同時に書き込み可能。スナップショットベースのイミュータブルな履歴が全イテレーションの痕跡を保持。ファイルレベルの競合検出を備えたワークスペースの3-wayマージ。システム `git` CLI ブリッジによる完全な git プロトコル互換性。クロスプラットフォーム（Linux、macOS）、crates.io に `libnoa` として公開。
