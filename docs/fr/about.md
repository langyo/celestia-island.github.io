« Les otakus de la tech sauvent le monde. »

**Celestia Island** est une organisation open source indépendante explorant les frontières de l'ingénierie système, ancrée dans le génie logiciel, recherchant et faisant progresser l'intelligence artificielle, les énergies nouvelles, l'IoT et les domaines connexes.

Chaque projet est conçu avec soin, repoussant les limites de ce que les logiciels système modernes peuvent accomplir.

Nos projets résonnent les uns avec les autres, chacun grandissant à son propre rythme.

*Cette organisation n'a aucun lien direct avec les sources d'inspiration des propriétés intellectuelles fictives.*
