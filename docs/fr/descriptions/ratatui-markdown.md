# Ratatui Markdown

Rendu de texte enrichi pour applications terminal, construit sur ratatui.

Rendu Markdown avec coloration syntaxique pour plus de 30 langages via tree-sitter, diagrammes Mermaid (séquence, camembert, Gantt, état, classe, flux), vues d'arbre JSON/TOML repliables et interactives, et un système de scroll hybride double mode. Tout sélectionnable via feature gates — n'incluez que ce dont votre app TUI a besoin.
