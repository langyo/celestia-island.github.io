# Hikari

Une bibliothèque de composants UI **native WASI** — des composants soignés et prêts à l'emploi qui s'exécutent sur le même runtime WASI que votre backend.

Construite sur Tairitsu, Hikari fournit un ensemble complet de composants UI sans la surcharge de `wasm-bindgen` ni de code glue JavaScript. Construisez vos interfaces une fois, affichez-les partout où le runtime WASI peut aller : navigateur, serveur ou edge.
