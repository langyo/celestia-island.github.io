# Aoba

Un compagnon de débogage de protocoles industriels. Inspectez, testez et simulez des appareils Modbus directement depuis votre terminal.

Connectez-vous à du matériel réel via des ports série, créez des appareils virtuels pour les tests, faites le pont entre les protocoles — Aoba est conçu pour les ingénieurs en automatisation travaillant aussi bien en atelier que dans les pipelines CI. Il comprend à la fois un tableau de bord TUI complet et une interface CLI scriptable.
