# Evernight

Un **courtier de contrôle à distance et de protocoles industriels** multiplateforme — le pont entre les agents IA cloud et le monde physique.

Evernight stream les écrans, pilote des sessions SSH/VNC/RDP, interroge la télémétrie matérielle et parle la pile de protocoles industriels — Modbus, S7comm, OPC UA, CAN, EtherCAT — via un runtime unifié. Il orchestre aussi conteneurs et VM en périphérie, si bien qu'un agent à l'étage peut atteindre un automate en atelier aussi naturellement qu'un appel de fonction local.
