# Shittim Chest

Le visage interactif de la plateforme multi-agent Entelecheia.

Shittim Chest, c'est tout ce que l'utilisateur voit et touche : une interface de chat en streaming pour converser avec les agents IA en temps réel, un panneau de contrôle pour configurer les systèmes et gérer les permissions, ainsi que des applications desktop et mobiles. C'est le pont entre l'intention humaine et l'action des agents.

Discutez avec les agents, générez des images, surveillez l'état des agents, configurez les contrôles d'accès — le tout via une interface web responsive et multilingue connectée au cœur d'agents d'Entelecheia.
