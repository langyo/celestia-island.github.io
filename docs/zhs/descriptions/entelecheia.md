# Entelecheia（玄枢）

基于 **harness engineering** 理念的多智能体平台，面向安全关键的工业控制。

不同于让 AI 模型无限制地调用工具，Entelecheia 将每个智能体严格约束于其专精领域——明确定义它能做什么、不能做什么。LLM 只能看到少数原语；每一次真正的动作都运行在 IEPL TypeScript 管线内，并跨多个智能体调度庞大的 MCP 工具面。

它的与众不同之处在于工业视角：跨厂商协议支持（Modbus、S7comm、OPC UA），从指令审查到人工审批的多层安全关卡，以及容器隔离执行。从控制室到车间现场，Entelecheia 替你操心协作细节，让智能体始终在边界之内行事——绝不越界。
