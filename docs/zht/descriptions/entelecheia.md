# Entelecheia（玄樞）

基於 **harness engineering** 理念的多智能體平台，面向安全關鍵的工業控制。

不同於讓 AI 模型無限制地呼叫工具，Entelecheia 將每個智能體嚴格約束於其專精領域——明確定義它能做什麼、不能做什麼。LLM 只能看到少數原語；每一次真正的動作都執行在 IEPL TypeScript 管線內，並跨多個智能體調度龐大的 MCP 工具面。

它的與眾不同之處在於工業視角：跨廠商協定支援（Modbus、S7comm、OPC UA），從指令審查到人工審批的多層安全關卡，以及容器隔離執行。從控制室到車間現場，Entelecheia 替你操心協作細節，讓智能體始終在邊界之內行事——絕不越界。
