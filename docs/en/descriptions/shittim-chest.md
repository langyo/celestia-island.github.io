# Shittim Chest

The interactive face of the Entelecheia multi-agent platform.

Shittim Chest is everything the user sees and touches: a streaming chat interface to converse with AI agents in real time, a control panel to configure systems and manage permissions, plus desktop and mobile applications. It is the bridge between human intent and agent action.

Chat with agents, generate images, monitor agent status, configure access controls — all through a responsive, multi-language web interface that connects to the Entelecheia agent core.
