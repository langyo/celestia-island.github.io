# Evernight

A cross-platform **remote control and industrial-protocol broker** — the bridge between cloud AI agents and the physical world.

Evernight streams screens and drives SSH, VNC, and RDP sessions, polls hardware telemetry, and speaks the industrial protocol stack — Modbus, S7comm, OPC UA, CAN, EtherCAT — through one unified runtime. It also orchestrates containers and VMs at the edge, so an agent upstairs can reach a PLC on the factory floor as naturally as a local function call.
