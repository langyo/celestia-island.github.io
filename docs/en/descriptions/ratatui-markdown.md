# Ratatui Markdown

Rich text rendering for terminal applications, built on ratatui.

Markdown rendering with syntax highlighting for 30+ languages via tree-sitter, Mermaid diagram rendering (sequence, pie, Gantt, state, class, flowchart), interactive collapsible JSON/TOML tree views, and a hybrid dual-mode scroll system. Everything is feature-gated — pull in only what your TUI app needs.
