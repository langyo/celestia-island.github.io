# Yuuka

Una invocación de macro, toda una jerarquía de tipos.

Biblioteca proc-macro que genera structs y enums complejos y profundamente anidados desde un DSL compacto. Los sub-structs y enums inline se aplanan automáticamente en definiciones Rust correctamente tipadas con integración serde. La macro auto! complementaria produce sintaxis de instanciación ergonómica — adiós a la construcción manual campo por campo.
