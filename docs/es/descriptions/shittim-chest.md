# Shittim Chest

La cara interactiva de la plataforma multi-agente Entelecheia.

Shittim Chest es todo lo que el usuario ve y toca: una interfaz de chat con streaming para conversar con agentes IA en tiempo real, un panel de control para configurar sistemas y gestionar permisos, además de aplicaciones de escritorio y móviles. Es el puente entre la intención humana y la acción de los agentes.

Chatea con agentes, genera imágenes, monitoriza el estado de los agentes, configura controles de acceso — todo a través de una interfaz web responsive y multilingüe que se conecta al núcleo de agentes de Entelecheia.
