# seia

Una biblioteca y CLI de **búsqueda web multi-motor** — una consulta, muchos backends, resultados unificados.

seia ofrece una interfaz única a una lista creciente de proveedores de búsqueda. Los motores que no requieren autenticación funcionan de inmediato sin configuración, de modo que puedes enrutar una consulta hacia varios backends y fusionar los resultados sin tener que escribir un cliente por motor cada vez.
