# Noa

Documentación para crates Rust, generada desde el fuente.

Un generador de documentación que analiza código fuente Rust y comentarios de documentación para producir páginas Markdown, sitios HTML estáticos y traducciones multilingües potenciadas por IA. Funciones planificadas incluyen caché basado en RON, generación incremental e integración con herramientas de documentación existentes.
