# Ratatui Markdown

Renderizado de texto enriquecido para aplicaciones de terminal, construido sobre ratatui.

Renderizado Markdown con resaltado de sintaxis para más de 30 lenguajes vía tree-sitter, diagramas Mermaid (secuencia, pastel, Gantt, estado, clase, flujo), vistas de árbol JSON/TOML colapsables e interactivas, y un sistema de scroll híbrido de doble modo. Todo seleccionable mediante feature gates — incluye solo lo que tu app TUI necesita.
