# KEI

Un **fork ARM64 independiente de asterinas**, que proporciona el binario de kernel consumido por ARIS.

KEI no es una rama de seguimiento. Absorbe el asterinas upstream según su propio calendario, añadiendo soporte ARM64 y Board Support Packages a medida para las pasarelas IoT industriales. El mismo modelo de fork que Apple emplea para LLVM — independencia con absorción upstream periódica — aplicado a un kernel de sistema operativo construido para la planta.
