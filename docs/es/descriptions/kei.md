# KEI

**Un núcleo de SO en Rust para IoT industrial** — derivado de Asterinas, en tiempo real y auditable.

KEI es un núcleo de SO en Rust para dispositivos edge ARM64 y RISC-V, derivado de Asterinas y moldeado en una instalación en tiempo real y auditable, con soporte de placas ARM64, pantalla virtio-gpu y controladores industriales. También incluye una biblioteca no_std para nodos sensores embassy, manteniéndose independiente de los ciclos de publicación originales.
