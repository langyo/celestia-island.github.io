# Lagrange

Una facilidad de sitio estático Markdown **renderizada por WASI** — multilingüe desde el inicio.

Entrégale a Lagrange una carpeta de Markdown y te devuelve un sitio de documentación estático y multilingüe, con selector de idioma integrado y redirección raíz. Sin framework JavaScript, sin mdBook, sin toolchain de Node. Renderiza su propia documentación — los docs que publicas fueron producidos por el propio Lagrange.
