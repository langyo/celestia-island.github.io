# Aoba

Un compañero de depuración de protocolos industriales. Inspecciona, prueba y simula dispositivos Modbus directamente desde tu terminal.

Conéctate a hardware real a través de puertos serie, crea dispositivos virtuales para pruebas, conecta datos entre protocolos — Aoba está diseñado para ingenieros de automatización que trabajan tanto en planta como en pipelines CI. Incluye tanto un panel TUI completo como una interfaz CLI programable.
