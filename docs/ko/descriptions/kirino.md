# Kirino

**제로 트러스트 인증·인가** — 단순한 RBAC를 넘어 — Rust 생태계에 도입하려는 원칙적인 프레임워크.

Kirino는 ANSI INCITS 359-2004 RBAC를 완전히 구현합니다(RBAC0/1/2). 직무 분리, 기수 제한, 시간적 제약, 역할 계층. 단순히 "누가 무엇을 할 수 있는지"가 아니라 "어떤 조건에서 가능한지"를 묻습니다.

그 위에 신뢰 점수 책정, 이상 탐지, DO-178C에서 영감받은 자율 등급(L0–L4)을 갖춘 동적 인가 레이어가 겹칩니다 — 접근 결정은 정적인 Yes/No가 아니라 리스크에 적응합니다. 실세계의 인가를, 애플리케이션에 직접 임베드할 수 있는 컴포저블 Rust 모듈로 증류했습니다.
