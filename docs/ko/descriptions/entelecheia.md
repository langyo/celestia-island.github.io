# Entelecheia

**하네스 엔지니어링** 원칙에 기반한 다중 에이전트 협업 플랫폼.

AI 모델에 무제한 도구 접근을 허용하는 대신, Entelecheia는 각 에이전트를 해당 도메인으로 제한합니다 — 무엇을 할 수 있고 무엇을 할 수 없는지 정확히 정의합니다. 이 하네스 엔지니어링 접근 방식은 에이전트가 무한한 자유 속에서 방황하기보다 경계 내에서 탁월함을 발휘하도록 합니다.

전문 에이전트 팀을 구성하고 목표를 설명하면, 에이전트들이 분업하고 협력하여 결과를 도출합니다. 연구 및 분석에서 코드 생성, 워크플로 자동화에 이르기까지, Entelecheia가 조정을 담당하므로 사용자는 결과에 집중할 수 있습니다.
