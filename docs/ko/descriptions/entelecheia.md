# Entelecheia

**하네스 엔지니어링** 원칙에 기반하여 안전이 중요한 산업 제어를 위해 구축된 다중 에이전트 플랫폼.

AI 모델에 무제한 도구 접근을 허용하는 대신, Entelecheia는 각 에이전트를 해당 도메인으로 제한합니다 — 무엇을 할 수 있고 무엇을 할 수 없는지 정확히 정의합니다. LLM이 보는 것은 소수의 프리미티브뿐입니다. 모든 실제 동작은 IEPL TypeScript 파이프라인 안에서 실행되어 수많은 에이전트에 걸친 방대한 MCP 도구 영역으로 디스패치됩니다.

이를 특별하게 만드는 것은 산업 관점입니다. 크로스 벤더 프로토콜 지원(Modbus, S7comm, OPC UA), 명령 검토부터 사람의 승인까지의 다층 안전 게이트, 그리고 컨테이너 격리 실행. 제어실에서 현장까지, Entelecheia가 조정을 담당하므로 에이전트는 항상 경계 안에서 행동하며 — 결코 넘지 않습니다.
