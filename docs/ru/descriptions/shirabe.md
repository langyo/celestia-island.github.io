# shirabe

Лёгкая **Rust-нативная библиотека автоматизации браузера** и сервер отладки.

shirabe управляет любым браузером, говорящим на Chrome DevTools Protocol — Chrome, Chromium, Edge, — через один движок ручной работы. Зафиксированная сборка Chrome for Testing загружается в общий кеш и находится прозрачно, поэтому браузер вручную ставить не нужно. Сродни ort для ONNX Runtime, только для headless-браузера.
