# KEI

**Ядро ОС на Rust для промышленного IoT** — на базе Asterinas, реальное время и аудит.

KEI — ядро ОС на Rust для edge-устройств ARM64 и RISC-V, производное от Asterinas и превращённое в средство реального времени с аудитом, с поддержкой плат ARM64, дисплеем virtio-gpu и промышленными драйверами. Также включает библиотеку no_std для сенсорных узлов embassy, оставаясь независимым от циклов выпуска upstream.
