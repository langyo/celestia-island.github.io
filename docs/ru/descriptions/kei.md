# KEI

**Независимый ARM64-форк asterinas**, предоставляющий бинарник ядра, используемый ARIS.

KEI — не отслеживающая ветка. Он поглощает upstream-asterinas по собственному графику, добавляя поддержку ARM64 и Board Support Packages, подобранные под промышленные IoT-шлюзы. Та же модель форка, что Apple применяет к LLVM, — независимость с периодическим поглощением upstream, — только применённая к ядру ОС, созданному для цеха.
