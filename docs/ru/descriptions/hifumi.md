# Hifumi

Мигрируйте данные между версиями структур без бойлерплейта.

Аннотируйте текущую версию struct через #[version] и выразите изменения полей через #[migration("0.1" => "0.2" { ... })] — Hifumi автогенерирует все старые версии struct и impl From<Old> for New. Сопутствующий CLI сравнивает определения структур между коммитами git и выводит готовые аннотации миграции.
