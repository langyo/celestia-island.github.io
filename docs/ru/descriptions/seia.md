# seia

Библиотека и CLI **многомоторного веб-поиска** — один запрос, множество бэкендов, единый результат.

seia предлагает единый интерфейс к растущему списку поисковых провайдеров. Движки, не требующие аутентификации, работают сразу без настройки, так что вы можете маршрутизировать запрос к нескольким бэкендам и объединять результаты, не переписывая клиент под каждый движок каждый раз.
