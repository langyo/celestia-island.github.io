# Ichika

Объявите пайплайн — макрос соединит потоки.

Макрос pipe! принимает последовательность замыканий — синхронных или асинхронных, именованных или анонимных — и генерирует полный пул потоков со всей канальной проводкой. Ветвление через match направляет данные в разные ветки пайплайна. Политики повторов с настраиваемой задержкой. Восстановление ошибок, сохраняющее работу пула. Graceful shutdown при Drop.
