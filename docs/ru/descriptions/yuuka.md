# Yuuka

Один вызов макроса — вся иерархия типов.

Библиотека proc-macro, генерирующая сложные глубоко вложенные struct и enum из компактного DSL. Inline-подструктуры и enum автоматически разворачиваются в корректно типизированные Rust-определения с интеграцией serde. Сопутствующий макрос auto! генерирует эргономичный синтаксис инстанцирования — больше не нужно вручную конструировать вложенные типы поле за полем.
