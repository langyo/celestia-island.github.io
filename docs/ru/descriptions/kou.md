# Kou

Автономный **движок виртуального терминала** — управление PTY, эмулятор экрана VT100/ANSI и рендеринг глифов в одной библиотеке.

Извлечённое и усиленное из упаковщика tairitsu, kou пропускает поток байтов через настоящий парсер `vte`: перемещения курсора CSI, стирание, прокрутка и палитра SGR учитываются, а не отбрасываются. Это ядро vtty, превращающее «голый» PTY в настоящий терминал, который можно читать и отрисовывать.
