# Shittim Chest

Интерактивный интерфейс мульти-агентной платформы Entelecheia.

Shittim Chest — это всё, что пользователь видит и с чем взаимодействует: чат-интерфейс с потоковой передачей для общения с ИИ-агентами в реальном времени, панель управления для настройки систем и управления разрешениями, а также десктопные и мобильные приложения. Это мост между человеческим намерением и действиями агентов.

Общайтесь с агентами, генерируйте изображения, отслеживайте состояние агентов, настраивайте контроль доступа — всё через адаптивный мультиязычный веб-интерфейс, подключённый к агентному ядру Entelecheia.
