# ARIS

**Встраиваемая ОС** промышленных IoT-шлюзов Entelecheia — запускает evernight на периферийных устройствах ARM/RISC-V.

ARIS — это слой прошивки, опускающий брокер протоколов к физическому оборудованию: ПЛК, датчикам, клапанам, исполнительным механизмам. Через минимальное, защищённое ядро он открывает устройства ввода-вывода для evernight, удерживая поверхность атаки как можно меньше — последний прыжок между программным намерением и физическим действием.
