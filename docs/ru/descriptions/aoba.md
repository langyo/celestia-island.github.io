# Aoba

Инструмент для отладки промышленных протоколов. Проверяйте, тестируйте и моделируйте устройства Modbus прямо из терминала.

Подключайтесь к реальному оборудованию через последовательные порты, создавайте виртуальные устройства для тестирования, соединяйте данные между протоколами — Aoba создан для инженеров по автоматизации, работающих как на производстве, так и в CI-пайплайнах. Включает как полнофункциональную TUI-панель, так и скриптуемый интерфейс командной строки.
