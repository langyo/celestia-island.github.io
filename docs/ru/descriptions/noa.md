# Noa

Документация для crate'ов Rust, сгенерированная из исходного кода.

Генератор документации, анализирующий исходный код Rust и комментарии документации для создания Markdown-страниц, статических HTML-сайтов и мультиязычных переводов на базе ИИ. В планах — кэширование на базе RON, инкрементальная генерация и интеграция с существующими инструментами документации.
