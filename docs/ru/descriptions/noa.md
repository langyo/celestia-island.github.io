# Noa

Нативная для ИИ распределённая система контроля версий — git для исходного кода, noa для данных итераций ИИ-агентов.

Сосуществует с `.git` в одном рабочем дереве. Журналы JSONL только для добавления на каждого агента (без блокировок) позволяют сотням агентов писать одновременно. Неизменяемая история на основе снапшотов сохраняет полный след итераций. Трёхстороннее слияние рабочих пространств с обнаружением конфликтов на уровне файлов. Полная совместимость с протоколом git через системный мост `git` CLI. Кроссплатформенность (Linux, macOS), опубликован на crates.io как `libnoa`.
