# Lagrange

Инструмент статического Markdown-сайта с **рендерингом через WASI** — многоязычный с самого начала.

Дайте Lagrange папку с Markdown, и он вернёт статический, многоязычный сайт документации со встроенным переключателем языков и корневым редиректом. Без JavaScript-фреймворка, без mdBook, без Node-инструментария. Он рендерит собственную документацию — те документы, что вы публикуете, произведены самим Lagrange.
