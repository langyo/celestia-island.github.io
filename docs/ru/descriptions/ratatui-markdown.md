# Ratatui Markdown

Рендеринг обогащённого текста для терминальных приложений на базе ratatui.

Рендеринг Markdown с подсветкой синтаксиса для 30+ языков через tree-sitter, диаграммы Mermaid (последовательности, круговые, Ганта, состояний, классов, блок-схемы), интерактивные сворачиваемые деревья JSON/TOML и гибридная двухрежимная система прокрутки. Всё подключается через feature gates — берите только то, что нужно вашему TUI-приложению.
